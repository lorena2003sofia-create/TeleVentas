# Museo
Proyecto de gestión de obras de arte para el museo, desarrollado en Python usando Programación Orientada a Objetos.

## Funcionalidades
- Gestión del catálogo de obras de arte (cuadros, esculturas y otros objetos)
- Registro de restauraciones de obras
- Control de cesión de obras a otros museos
- Valoración total de todas las obras del museo
- Autenticación de usuarios según rol (director, restaurador, visitante)

## Ejecutar
python main.py
