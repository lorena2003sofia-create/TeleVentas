# Taller Programación Orientada a Objetos

Este repositorio contiene los ejercicios desarrollados para el taller de Programación Orientada a Objetos.

Los proyectos fueron desarrollados en Python aplicando conceptos de clases, objetos, encapsulamiento y principios básicos SOLID.

## Estructura del repositorio

TeleVentas/
Contiene el ejercicio del sistema de compras a distancia.
Incluye catálogo de productos, órdenes de compra y gestión de clientes.

Museo/
Contiene el ejercicio del sistema de gestión de obras de arte.
Incluye manejo de obras, restauraciones y control de préstamos entre museos.

## Conceptos aplicados

- Programación Orientada a Objetos
- Clases y objetos
- Métodos y atributos
- Organización modular del código
- Principio de responsabilidad única
